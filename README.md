## Gestion des dates

Le format de date utilisé est le format ISO 8601 ("YYYY-MM-DD").
Lors de l'initialisation ou de la modification de la date de parution d'un média, toute valeur respectant le format attendu est affectée, sinon la date sera le premier jour du mois en cours.
Cette vérification est mise en place pour garder la cohérence des dates stockées.
